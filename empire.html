Total War: EMPIRE

UX Engineer for IOS and Android

Feral Interactive is a British video games developer and publisher for macOS, Linux, iOS, Android and Nintendo Switch. It was founded in 1996 with a focus on bringing games to Mac, it now specialises in porting games to mobile platforms.

I worked with Feral Interactive as a UX Engineer. The duration of my time here was spent as a lead UX Engineer on Total War: EMPIRE. I used software such as Figma here to design various screens. I coded the interface using a HTML and CSS based language which is compatible with C++.

I considered a wide range of devices for the development such as many iPhones and iPads aswell as various Android phones and tablets. This meant my approach to creating the game's UI had to be dynamic and responsive. For more information on this see Feral Interactive's Empire page.

My work spanned across different types of UI for the game - for the campain screens, battle screens and menu screens.
