Railroads!

Front-end developer for IOS and Android.

I worked with Feral Interactive as a UX Engineer, one of my projects was the mobile port of the classic train tycoon game Sid Meier's Railroads! to IOS and Android. I coded the interface using RML, a HTML based library which is compatible with C++.

When coding an interface for mobile devices it has a similar process and problem as web pages, that being a user could be using a screen of all shapes and sizes! From iphone SE, iphone 14 Pro Max to the iPad Air 4, my UI had to be modular and dynamic.

Another issue here was UI bugs cause by localisation, languages such as Russian, Japanese, Spanish and Italian have vastly different lengths for words than English. Therefore various buttons and menus had to be ready and dynamic for strings of any possible length.

My work here gave me vast a knowledge in UI development and bug-fixing.
