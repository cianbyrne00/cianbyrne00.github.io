Planet Sustainability.

An Interactive Educational Installation Made in Unreal Engine 5.

For my M.Sc. in Interactive Digital Media at Trinity College Dublin I worked on a three month research project which explored the realms of digital media. My group decided to create an interactive installation which used the user's body as a controller through motion sensor technology. We called it Planet Sustainability.

Planet Sustainability is an interactive installation made with the Unreal Engine 5 game development engine. The project aims to create an experience which engages and educates the user on the topic of Environmentalism in daily choices.

You experience the installation from a first-person perspective, navigating the 3D worlds we made with your body using the Microsoft Kinect as a controller (with a traditional input method such as Keyboard and mouse also supported). The installation features 4-way surround sound for audio using mounted speakers on stands and a 1080p projector to display the video output.

The goal here is a covid conscious experience that is fun, immersive, engaging and educational. The 3D-conscious audio design, impressive graphics, and body controls make the project unique.

My role in this project was varied as I dabbled accross project management, 3D graphics and programming. I created many 3D models for this project and textured them. I created audio FX, sourced audio FX and created all music myself for this project.
