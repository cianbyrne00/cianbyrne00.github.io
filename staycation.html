Staycation.ie

Hotel Booking UX Design

For my professional diploma with the UX Design Institute, my final project had to be an interactive prototype of a Hotel Booking service.

I decided to create one around the theme of staycations in Ireland, using a green gradient across the design to reflect this concept.

This project followed a strict design process. Begining with research, user interviews and testing, affinity diagrams and finally wireframes and prototyping. Find the interactive prototype embedded below.
